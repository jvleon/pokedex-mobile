# Pokedex mobile

- Tested only on Android.
- List Pokemons and berries.

### Requirements:

- Android studio
- node 10>=
- android SDK
- java SDK
- git
- react-native cli or npx
- npm or yarn

### Setup:

- `git clone https://github.com/jvleon/pokedex-mobile.git && cd pokedex-mobile`
- `npm install` or `yarn`
- `npx react-native start`
- Open a new terminal instance and go to the repository directory.
- `npx react-native run-android --variant=debug`
